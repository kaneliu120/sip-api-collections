## How to Use

### Option 1: Apify Console (no code)
Visit any Actor link above, click **Start**, get results in the dataset tab.

### Option 2: API
```bash
curl -X POST "https://api.apify.com/v2/acts/lentic_clockss~global-sanctions-screening/run-sync-get-dataset-items" \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{"searchTerms": ["Rosneft"]}'
```

### Option 3: Direct SIP API
```bash
curl -H "X-API-Key: YOUR_KEY" \
  "https://sip.myskillstore.dev/api/v1/data/products/us_ofac_sanctions/search?q=Rosneft&limit=10"
```

### Option 4: Postman
[Fork the collection](https://www.postman.com/kaneliu10/sip-data-products-api) — pre-built requests with example responses included.
